# Background

This folder contains versions of the contents of `/scripts` that run the scripts using a GUI rather than the command line.

## Goals

The goal of this is to:

* Manually navigate to a directory;
* Manually select the source file
  * Limit the files that can be selected
* Implement proper logging
* Launch without requiring the command line

Once complete, the GUIs will make it easier to run these scripts for anyone regardless of technical proficiency
